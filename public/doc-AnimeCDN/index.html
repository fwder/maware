AnimeCDN 使用文档

AnimeCDN 介绍

这是什么项目

AnimeCDN 是基于 Github 代码托管库 + Vercel 中转节点 的番剧存储服务。

使用 Github 存储库

项目存储在 Github 近乎无限的存储空间（对于个人用户来说，单文件不超过100M、单个库不超过100G），所以 AnimeCDN 可以存储大量番剧资源并保证在较长时间内不会丢失。

这是一个错误的使用示例。详见官方文档 虽然 Github 提供近乎无限的资源，但由于 Git 始终不适用于大文件存储，您也不应该将文件存储服务托管在 Github 上。

我们建议每个仓库均保持在 1GB 以下。 仓库的硬限制为 100GB。 如果达到 75GB，推送时您将在终端收到 Git 的警告。 如果将大文件留在仓库之外，则此限制很容易保持。 如果仓库超过 1GB，您可能会收到来自 GitHub 支持的礼貌电子邮件，要求您减小仓库的大小以使其恢复。

因此，我们将会使用更多的库来满足空间需求（您将会在 AnimeCDN 项目中看到很多作为分节点的库），并且您将看到所有超过100M的大文件都被拆分成多个文件储存在库中，您可能需要调整您的程序逻辑来适配这种规则。

使用 Vercel 作为中转节点

由于 Github 资源服务器在中国被墙，在没有魔法上网的前提下是不能访问资源地址的，所以我们使用了 Vercel 的路由重写功能来为我们搭建中转节点（Vercel 同样有限制，普通用户限制每月200G的流量）。

您也可以使用您自己搭建的中转节点来代理。

如果您有自己的服务器或者Vercel账号，我们强烈建议您使用自己搭建的节点来进行中转。（详见后文教程）（官方节点撑不住这么多人艹啊啊啊）
